Ender Dragon

The Ender dragon is the strongest mob in minecraft because it can get healed and respawned. It also deliers most damage per hit. Lastly the Ender Dragon releases a poison out of its mouth that can kill you

Iron Golem

An Iron Glemn is a mob that you can create by using 3 iron blocks and 1 pumpkin for his head. Iron golems only spawn in villages and usually likes the villagers. It can kill with 2 hits but It only attacks if you are not cooperating with villagers or if you attack it.

Endermen

Endermen are one of the scariest mobs becaause of the fact that it can move objscts and teleport. Because they teleport you can't hit it with a bow. Once you hit one you have to kill it or jump in water because endermen can't touch water. Also, If you run from it It will easily teleport right behind you.

Ghast

A Ghast Is a mob that can be only found in the nether, the nether is a world that can be went to using a portal. The ghast can blow flaming rocks at pllayers which can do a deadly amount of damage if it hits. You can kill it with a sword but it is hard to get them near you so you should use a bow.

Creeper

A creeper is a mob found in the original world in caves and on the grass and mountains at night. the creeper explodes if you come too close if you manage to kill it before it explodes you can get gunpowder which is useful for making tnt
